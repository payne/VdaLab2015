Getting started

  1. Use$ p5 new COLLECTION_NAME to create a collection
  2. Then$ p5 generate PROJECT_NAMEto generate a project
  3. Run$ p5 serverin the collection directory
  4. Go tolocalhost:5555and you should see this page
  5. All your projects would be listed in the left
